# 股票分析模块接口设计

## 1. 接口列表

| 接口路径 | 方法 | 功能描述 | 请求参数 | 响应数据 | 权限要求 |
|---------|------|---------|---------|---------|----------|
| `/api/v1/stock/search` | GET | 搜索股票 | keyword: str (必填，长度1-50) | `[{"code": str, "name": str, "market": str}]` | 已登录 |
| `/api/v1/stock/{code}` | GET | 获取股票基本信息 | code: str (路径参数，必填) | `{"code": str, "name": str, "market": str, "industry": str, "pe": float, "pb": float, "market_cap": float}` | 已登录 |
| `/api/v1/stock/{code}/price` | GET | 获取股票价格数据 | code: str (路径参数，必填)<br>start_date: str (必填，YYYY-MM-DD)<br>end_date: str (必填，YYYY-MM-DD) | `{"code": str, "name": str, "data": [{"date": str, "open": float, "high": float, "low": float, "close": float, "volume": int}]}` | 已登录 |
| `/api/v1/stock/{code}/financial` | GET | 获取股票财务数据 | code: str (路径参数，必填)<br>year: int (必填，4位数字) | `{"code": str, "name": str, "year": int, "revenue": float, "profit": float, "roe": float, "debt_ratio": float}` | 已登录 |
| `/api/v1/stock/{code}/analysis` | POST | 分析股票 | code: str (路径参数，必填)<br>analysis_type: str (必填，"fundamental"/"technical"/"comprehensive") | `{"code": str, "name": str, "analysis_type": str, "result": {"score": int, "conclusion": str, "details": {}}}` | 已登录 |
| `/api/v1/stock/{code}/news` | GET | 获取股票相关新闻 | code: str (路径参数，必填)<br>page: int (默认1，最小值1)<br>page_size: int (默认10，最小值1，最大值50) | `{"code": str, "name": str, "news": [{"title": str, "content": str, "date": str, "source": str}]}` | 已登录 |

## 2. 接口详情

### 2.1 搜索股票

**接口路径**：`/api/v1/stock/search`
**方法**：GET
**功能描述**：搜索股票
**请求参数**：
- keyword: str (必填，长度1-50)

**响应数据**：
```json
{
  "code": 200,
  "message": "搜索成功",
  "data": [
    {
      "code": "600000",
      "name": "浦发银行",
      "market": "SH"
    },
    {
      "code": "600036",
      "name": "招商银行",
      "market": "SH"
    }
  ]
}
```

### 2.2 获取股票基本信息

**接口路径**：`/api/v1/stock/{code}`
**方法**：GET
**功能描述**：获取股票基本信息
**请求参数**：
- code: str (路径参数，必填)

**响应数据**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": {
    "code": "600000",
    "name": "浦发银行",
    "market": "SH",
    "industry": "银行",
    "pe": 5.8,
    "pb": 0.6,
    "market_cap": 280000000000
  }
}
```

### 2.3 获取股票价格数据

**接口路径**：`/api/v1/stock/{code}/price`
**方法**：GET
**功能描述**：获取股票价格数据
**请求参数**：
- code: str (路径参数，必填)
- start_date: str (必填，YYYY-MM-DD)
- end_date: str (必填，YYYY-MM-DD)

**响应数据**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": {
    "code": "600000",
    "name": "浦发银行",
    "data": [
      {
        "date": "2026-01-01",
        "open": 10.0,
        "high": 10.2,
        "low": 9.8,
        "close": 10.1,
        "volume": 10000000
      },
      {
        "date": "2026-01-02",
        "open": 10.1,
        "high": 10.3,
        "low": 10.0,
        "close": 10.2,
        "volume": 12000000
      }
    ]
  }
}
```

### 2.4 获取股票财务数据

**接口路径**：`/api/v1/stock/{code}/financial`
**方法**：GET
**功能描述**：获取股票财务数据
**请求参数**：
- code: str (路径参数，必填)
- year: int (必填，4位数字)

**响应数据**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": {
    "code": "600000",
    "name": "浦发银行",
    "year": 2025,
    "revenue": 100000000000,
    "profit": 30000000000,
    "roe": 15.5,
    "debt_ratio": 85.2
  }
}
```

### 2.5 分析股票

**接口路径**：`/api/v1/stock/{code}/analysis`
**方法**：POST
**功能描述**：分析股票
**请求参数**：
- code: str (路径参数，必填)
- analysis_type: str (必填，"fundamental"/"technical"/"comprehensive")

**响应数据**：
```json
{
  "code": 200,
  "message": "分析成功",
  "data": {
    "code": "600000",
    "name": "浦发银行",
    "analysis_type": "comprehensive",
    "result": {
      "score": 85,
      "conclusion": "浦发银行基本面稳健，技术面呈现上升趋势，综合评级为买入。",
      "details": {
        "fundamental": {
          "score": 80,
          "metrics": {
            "pe": "合理",
            "pb": "低估",
            "roe": "良好"
          }
        },
        "technical": {
          "score": 90,
          "indicators": {
            "ma": "金叉",
            "macd": "多头",
            "kdj": "超买"
          }
        }
      }
    }
  }
}
```

### 2.6 获取股票相关新闻

**接口路径**：`/api/v1/stock/{code}/news`
**方法**：GET
**功能描述**：获取股票相关新闻
**请求参数**：
- code: str (路径参数，必填)
- page: int (默认1，最小值1)
