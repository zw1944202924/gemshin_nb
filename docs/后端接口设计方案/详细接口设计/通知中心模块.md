# 通知中心模块接口设计

## 1. 接口列表

| 接口路径 | 方法 | 功能描述 | 请求参数 | 响应数据 | 权限要求 |
|---------|------|---------|---------|---------|----------|
| `/api/v1/notification` | GET | 获取通知列表 | page: int (默认1，最小值1)<br>page_size: int (默认10，最小值1，最大值50)<br>status: str (可选，"unread"/"read") | `{"total": int, "notifications": [{"id": int, "title": str, "content": str, "type": str, "created_at": str, "status": str}]}` | 已登录 |
| `/api/v1/notification/{id}` | PUT | 标记通知为已读 | id: int (路径参数，必填) | `{"message": "通知已标记为已读"}` | 已登录 |
| `/api/v1/notification/read-all` | PUT | 标记所有通知为已读 | 无 | `{"message": "所有通知已标记为已读"}` | 已登录 |
| `/api/v1/notification/{id}` | DELETE | 删除通知 | id: int (路径参数，必填) | `{"message": "通知删除成功"}` | 已登录 |

## 2. 接口详情

### 2.1 获取通知列表

**接口路径**：`/api/v1/notification`
**方法**：GET
**功能描述**：获取通知列表
**请求参数**：
- page: int (默认1，最小值1)
- page_size: int (默认10，最小值1，最大值50)
- status: str (可选，"unread"/"read")

**响应数据**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": {
    "total": 25,
    "notifications": [
      {
        "id": 1,
        "title": "股票达到目标价",
        "content": "您关注的浦发银行(600000)已达到目标价11.0元",
        "type": "price_alert",
        "created_at": "2026-01-01 10:00:00",
        "status": "unread"
      },
      {
        "id": 2,
        "title": "分析报告生成完成",
        "content": "您的浦发银行综合分析报告已生成完成",
        "type": "report",
        "created_at": "2026-01-01 09:00:00",
        "status": "read"
      }
    ]
  }
}
```

### 2.2 标记通知为已读

**接口路径**：`/api/v1/notification/{id}`
**方法**：PUT
**功能描述**：标记通知为已读
**请求参数**：
- id: int (路径参数，必填)

**响应数据**：
```json
{
  "code": 200,
  "message": "通知已标记为已读",
  "data": {}
}
```

### 2.3 标记所有通知为已读

**接口路径**：`/api/v1/notification/read-all`
**方法**：PUT
**功能描述**：标记所有通知为已读
**请求参数**：无
**响应数据**：
```json
{
  "code": 200,
  "message": "所有通知已标记为已读",
  "data": {}
}
```

### 2.4 删除通知

**接口路径**：`/api/v1/notification/{id}`
**方法**：DELETE
**功能描述**：删除通知
**请求参数**：
- id: int (路径参数，必填)

**响应数据**：
```json
{
  "code": 200,
  "message": "通知删除成功",
  "data": {}
}
```
