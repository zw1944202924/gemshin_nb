# 持仓管理模块接口设计

## 1. 接口列表

| 接口路径 | 方法 | 功能描述 | 请求参数 | 响应数据 | 权限要求 |
|---------|------|---------|---------|---------|----------|
| `/api/v1/portfolio` | GET | 获取用户持仓列表 | 无 | `[{"id": int, "code": str, "name": str, "quantity": int, "cost_price": float, "current_price": float, "profit": float, "profit_rate": float}]` | 已登录 |
| `/api/v1/portfolio` | POST | 添加持仓 | code: str (必填)<br>name: str (必填)<br>quantity: int (必填，最小值1)<br>cost_price: float (必填，最小值0.01) | `{"id": int, "code": str, "name": str, "quantity": int, "cost_price": float}` | 已登录 |
| `/api/v1/portfolio/{id}` | PUT | 更新持仓 | id: int (路径参数，必填)<br>quantity: int (必填，最小值1)<br>cost_price: float (必填，最小值0.01) | `{"id": int, "code": str, "name": str, "quantity": int, "cost_price": float}` | 已登录 |
| `/api/v1/portfolio/{id}` | DELETE | 删除持仓 | id: int (路径参数，必填) | `{"message": "持仓删除成功"}` | 已登录 |
| `/api/v1/portfolio/analysis` | GET | 分析持仓组合 | 无 | `{"total_value": float, "total_profit": float, "total_profit_rate": float, "industry_distribution": {}, "risk_level": str, "suggestions": []}` | 已登录 |

## 2. 接口详情

### 2.1 获取用户持仓列表

**接口路径**：`/api/v1/portfolio`
**方法**：GET
**功能描述**：获取用户持仓列表
**请求参数**：无
**响应数据**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": [
    {
      "id": 1,
      "code": "600000",
      "name": "浦发银行",
      "quantity": 1000,
      "cost_price": 10.0,
      "current_price": 10.5,
      "profit": 500,
      "profit_rate": 5.0
    },
    {
      "id": 2,
      "code": "600036",
      "name": "招商银行",
      "quantity": 500,
      "cost_price": 30.0,
      "current_price": 32.0,
      "profit": 1000,
      "profit_rate": 6.67
    }
  ]
}
```

### 2.2 添加持仓

**接口路径**：`/api/v1/portfolio`
**方法**：POST
**功能描述**：添加持仓
**请求参数**：
- code: str (必填)
- name: str (必填)
- quantity: int (必填，最小值1)
- cost_price: float (必填，最小值0.01)

**响应数据**：
```json
{
  "code": 200,
  "message": "添加成功",
  "data": {
    "id": 3,
    "code": "601318",
    "name": "中国平安",
    "quantity": 200,
    "cost_price": 45.0
  }
}
```

### 2.3 更新持仓

**接口路径**：`/api/v1/portfolio/{id}`
**方法**：PUT
**功能描述**：更新持仓
**请求参数**：
- id: int (路径参数，必填)
- quantity: int (必填，最小值1)
- cost_price: float (必填，最小值0.01)

**响应数据**：
```json
{
  "code": 200,
  "message": "更新成功",
  "data": {
    "id": 1,
    "code": "600000",
    "name": "浦发银行",
    "quantity": 1500,
    "cost_price": 10.2
  }
}
```

### 2.4 删除持仓

**接口路径**：`/api/v1/portfolio/{id}`
**方法**：DELETE
**功能描述**：删除持仓
**请求参数**：
- id: int (路径参数，必填)

**响应数据**：
```json
{
  "code": 200,
  "message": "持仓删除成功",
  "data": {}
}
```

### 2.5 分析持仓组合

**接口路径**：`/api/v1/portfolio/analysis`
**方法**：GET
**功能描述**：分析持仓组合
**请求参数**：无
**响应数据**：
```json
{
  "code": 200,
  "message": "分析成功",
  "data": {
    "total_value": 85000,
    "total_profit": 3500,
    "total_profit_rate": 4.32,
    "industry_distribution": {
      "银行": 60,

    },
    "risk_level": "中等",
    "suggestions": [
      "建议增加科技行业的配置，降低行业集中度",
      "建议设置止损位，控制风险"
    ]
  }

```
