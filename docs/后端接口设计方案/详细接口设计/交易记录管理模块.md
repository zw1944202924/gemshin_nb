## 2. 接口详情

### 2.1 获取交易记录

**接口路径**：`/api/v1/transaction`
**方法**：GET
**功能描述**：获取交易记录
**请求参数**：
- page: int (默认1，最小值1)
- page_size: int (默认10，最小值1，最大值50)
- start_date: str (可选，YYYY-MM-DD)
- end_date: str (可选，YYYY-MM-DD)

**响应数据**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": {
    "total": 50,
    "transactions": [
      {
        "id": 1,
        "code": "600000",
        "name": "浦发银行",
        "type": "buy",
        "quantity": 1000,
        "price": 10.0,
        "amount": 10000,
        "date": "2026-01-01"
      },
      {
        "id": 2,
        "code": "600036",
        "name": "招商银行",
        "type": "buy",
        "quantity": 500,
        "price": 30.0,
        "amount": 15000,
        "date": "2026-01-02"
      }
    ]
  }
}
```

### 2.2 添加交易记录

**接口路径**：`/api/v1/transaction`
**方法**：POST
**功能描述**：添加交易记录
**请求参数**：
- code: str (必填)
- name: str (必填)
- type: str (必填，"buy"/"sell")
- quantity: int (必填，最小值1)
- price: float (必填，最小值0.01)
- date: str (必填，YYYY-MM-DD)

**响应数据**：
```json
{
  "code": 200,
  "message": "添加成功",
  "data": {
    "id": 3,
    "code": "601318",
    "name": "中国平安",
    "type": "buy",
    "quantity": 200,
    "price": 45.0,
    "amount": 9000,
    "date": "2026-01-03"
  }
}
```

### 2.3 更新交易记录

**接口路径**：`/api/v1/transaction/{id}`
**方法**：PUT
**功能描述**：更新交易记录
**请求参数**：
- id: int (路径参数，必填)
- quantity: int (必填，最小值1)
- price: float (必填，最小值0.01)
- date: str (必填，YYYY-MM-DD)

**响应数据**：
```json
{
  "code": 200,
  "message": "更新成功",
  "data": {
    "id": 1,
    "code": "600000",
    "name": "浦发银行",
    "type": "buy",
    "quantity": 1500,
    "price": 10.2,
    "amount": 15300,
    "date": "2026-01-01"
  }
}
```

### 2.4 删除交易记录

**接口路径**：`/api/v1/transaction/{id}`
**方法**：DELETE
**功能描述**：删除交易记录
**请求参数**：
- id: int (路径参数，必填)

**响应数据**：
```json
{
  "code": 200,
  "message": "交易记录删除成功",
  "data": {}
}
```

### 2.5 获取交易统计

**接口路径**：`/api/v1/transaction/statistics`
**方法**：GET
**功能描述**：获取交易统计
**请求参数**：
- start_date: str (可选，YYYY-MM-DD)
- end_date: str (可选，YYYY-MM-DD)

**响应数据**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": {
    "total_buy": 34300,
    "total_sell": 25000,
    "total_profit": 3500,
    "win_rate": 75,
    "avg_holding_days": 30
  }
}
```
