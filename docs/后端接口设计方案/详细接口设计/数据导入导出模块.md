# 数据导入/导出模块接口设计

## 1. 接口列表

| 接口路径 | 方法 | 功能描述 | 请求参数 | 响应数据 | 权限要求 |
|---------|------|---------|---------|---------|----------|
| `/api/v1/data/import` | POST | 导入数据 | file: file (multipart/form-data，必填，支持Excel/CSV格式)<br>data_type: str (必填，"portfolio"/"transaction"/"watchlist") | `{"message": "数据导入成功", "count": int, "failed_count": int, "errors": []}` | 已登录 |
| `/api/v1/data/export` | GET | 导出数据 | data_type: str (必填，"portfolio"/"transaction"/"watchlist")<br>format: str (必填，"excel"/"csv") | 文件流 | 已登录 |

## 2. 接口详情

### 2.1 导入数据

**接口路径**：`/api/v1/data/import`
**方法**：POST
**功能描述**：导入数据
**请求参数**：
- file: file (multipart/form-data，必填，支持Excel/CSV格式)
- data_type: str (必填，"portfolio"/"transaction"/"watchlist")

**响应数据**：
```json
{
  "code": 200,
  "message": "数据导入成功",
  "data": {
    "count": 10,
    "failed_count": 2,
    "errors": [
      {
        "row": 3,
        "error": "股票代码格式不正确"
      },
      {
        "row": 7,
        "error": "数量必须为整数"
      }
    ]
  }
}
```

### 2.2 导出数据

**接口路径**：`/api/v1/data/export`
**方法**：GET
**功能描述**：导出数据
**请求参数**：
- data_type: str (必填，"portfolio"/"transaction"/"watchlist")
- format: str (必填，"excel"/"csv")

**响应数据**：
- 文件流（Excel或CSV文件）

## 3. 数据格式说明

### 3.1 持仓数据 (portfolio) 格式

**Excel/CSV字段**：
- `code`：股票代码（必填）
- `name`：股票名称（必填）
- `quantity`：持有数量（必填，整数）
- `cost_price`：成本价（必填，数值）

### 3.2 交易记录 (transaction) 格式

**Excel/CSV字段**：
- `code`：股票代码（必填）
- `name`：股票名称（必填）
- `type`：交易类型（必填，"buy"或"sell"）
- `quantity`：交易数量（必填，整数）
- `price`：交易价格（必填，数值）
- `date`：交易日期（必填，YYYY-MM-DD格式）

### 3.3 关注列表 (watchlist) 格式

**Excel/CSV字段**：
- `code`：股票代码（必填）
- `name`：股票名称（必填）
- `target_price`：目标价格（必填，数值）

## 4. 导入/导出示例

### 4.1 导入示例

```bash
# 使用curl导入持仓数据
curl -X POST "http://localhost:5000/api/v1/data/import" \
  -H "Authorization: Bearer {access_token}" \
  -F "file=@portfolio.xlsx" \
  -F "data_type=portfolio"
```

### 4.2 导出示例

```bash
# 使用curl导出交易记录
curl -X GET "http://localhost:5000/api/v1/data/export?data_type=transaction&format=excel" \
  -H "Authorization: Bearer {access_token}" \
  -o transactions.xlsx
```
