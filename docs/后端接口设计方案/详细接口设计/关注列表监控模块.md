# 关注列表监控模块接口设计

## 1. 接口列表

| 接口路径 | 方法 | 功能描述 | 请求参数 | 响应数据 | 权限要求 |
|---------|------|---------|---------|---------|----------|
| `/api/v1/watchlist` | GET | 获取用户关注列表 | 无 | `[{"id": int, "code": str, "name": str, "target_price": float, "current_price": float, "status": str}]` | 已登录 |
| `/api/v1/watchlist` | POST | 添加关注股票 | code: str (必填)<br>name: str (必填)<br>target_price: float (必填，最小值0.01) | `{"id": int, "code": str, "name": str, "target_price": float}` | 已登录 |
| `/api/v1/watchlist/{id}` | PUT | 更新关注股票 | id: int (路径参数，必填)<br>target_price: float (必填，最小值0.01) | `{"id": int, "code": str, "name": str, "target_price": float}` | 已登录 |
| `/api/v1/watchlist/{id}` | DELETE | 删除关注股票 | id: int (路径参数，必填) | `{"message": "关注股票删除成功"}` | 已登录 |
| `/api/v1/watchlist/monitor` | GET | 监控关注列表 | 无 | `[{"code": str, "name": str, "current_price": float, "target_price": float, "status": str, "alert": bool}]` | 已登录 |

## 2. 接口详情

### 2.1 获取用户关注列表

**接口路径**：`/api/v1/watchlist`
**方法**：GET
**功能描述**：获取用户关注列表
**请求参数**：无
**响应数据**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": [
    {
      "id": 1,
      "code": "600000",
      "name": "浦发银行",
      "target_price": 11.0,
      "current_price": 10.5,
      "status": "接近目标价"
    },
    {
      "id": 2,
      "code": "600036",
      "name": "招商银行",
      "target_price": 35.0,
      "current_price": 32.0,
      "status": "上涨中"
    }
  ]
}
```

### 2.2 添加关注股票

**接口路径**：`/api/v1/watchlist`
**方法**：POST
**功能描述**：添加关注股票
**请求参数**：
- code: str (必填)
- name: str (必填)
- target_price: float (必填，最小值0.01)

**响应数据**：
```json
{
  "code": 200,
  "message": "添加成功",
  "data": {
    "id": 3,
    "code": "601318",
    "name": "中国平安",
    "target_price": 50.0
  }
}
```

### 2.3 更新关注股票

**接口路径**：`/api/v1/watchlist/{id}`
**方法**：PUT
**功能描述**：更新关注股票
**请求参数**：
- id: int (路径参数，必填)
- target_price: float (必填，最小值0.01)

**响应数据**：
```json
{
  "code": 200,
  "message": "更新成功",
  "data": {
    "id": 1,
    "code": "600000",
    "name": "浦发银行",
    "target_price": 11.5
  }
}
```

### 2.4 删除关注股票

**接口路径**：`/api/v1/watchlist/{id}`
**方法**：DELETE
**功能描述**：删除关注股票
**请求参数**：
- id: int (路径参数，必填)

**响应数据**：
```json
{
  "code": 200,
  "message": "关注股票删除成功",
  "data": {}
}
```

### 2.5 监控关注列表

**接口路径**：`/api/v1/watchlist/monitor`
**方法**：GET
**功能描述**：监控关注列表
**请求参数**：无
**响应数据**：
```json
{
  "code": 200,
  "message": "监控成功",
  "data": [
    {
      "code": "600000",
      "name": "浦发银行",
      "current_price": 10.9,
      "target_price": 11.0,
      "status": "即将达到目标价",
      "alert": true
    },
    {
      "code": "600036",
      "name": "招商银行",
      "current_price": 32.0,
      "target_price": 35.0,
      "status": "上涨中",
      "alert": false
    }
  ]
}
```
