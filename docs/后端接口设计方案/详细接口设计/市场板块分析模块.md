# 市场板块分析模块接口设计

## 1. 接口列表

| 接口路径 | 方法 | 功能描述 | 请求参数 | 响应数据 | 权限要求 |
|---------|------|---------|---------|---------|----------|
| `/api/v1/industry` | GET | 获取行业列表 | 无 | `[{"code": str, "name": str, "weight": float, "change": float}]` | 已登录 |
| `/api/v1/industry/{code}` | GET | 获取行业详情 | code: str (路径参数，必填) | `{"code": str, "name": str, "weight": float, "change": float, "stocks": [{"code": str, "name": str, "weight": float, "change": float}]}` | 已登录 |
| `/api/v1/industry/analysis` | GET | 行业分析 | period: str (必填，"day"/"week"/"month"/"year") | `{"hot_industries": [{"code": str, "name": str, "change": float}], "cold_industries": [{"code": str, "name": str, "change": float}], "correlation": {}}` | 已登录 |

## 2. 接口详情

### 2.1 获取行业列表

**接口路径**：`/api/v1/industry`
**方法**：GET
**功能描述**：获取行业列表
**请求参数**：无
**响应数据**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": [
    {
      "code": "bank",
      "name": "银行",
      "weight": 20.5,
      "change": 1.5
    },
    {
      "code": "tech",
      "name": "科技",
      "weight": 15.2,
      "change": 2.3
    },
    {
      "code": "healthcare",
      "name": "医药",
      "weight": 10.8,
      "change": -0.5
    }
  ]
}
```

### 2.2 获取行业详情

**接口路径**：`/api/v1/industry/{code}`
**方法**：GET
**功能描述**：获取行业详情
**请求参数**：
- code: str (路径参数，必填)

**响应数据**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": {
    "code": "bank",
    "name": "银行",
    "weight": 20.5,
    "change": 1.5,
    "stocks": [
      {
        "code": "600000",
        "name": "浦发银行",
        "weight": 10.2,
        "change": 1.8
      },
      {
        "code": "600036",
        "name": "招商银行",
        "weight": 8.5,
        "change": 2.1
      },
      {
        "code": "601318",
        "name": "中国平安",
        "weight": 6.8,
        "change": 1.2
      }
    ]
  }
}
```

### 2.3 行业分析

**接口路径**：`/api/v1/industry/analysis`
**方法**：GET
**功能描述**：行业分析
**请求参数**：
- period: str (必填，"day"/"week"/"month"/"year")

**响应数据**：
```json
{
  "code": 200,
  "message": "分析成功",
  "data": {
    "hot_industries": [
      {
        "code": "tech",
        "name": "科技",
        "change": 5.2
      },
      {
        "code": "energy",
        "name": "能源",
        "change": 3.8
      }
    ],
    "cold_industries": [
      {
        "code": "real_estate",
        "name": "房地产",
        "change": -2.1
      },
      {
        "code": "consumer",
        "name": "消费品",
        "change": -1.5
      }
    ],
    "correlation": {
      "bank": [
        {"industry": "finance", "correlation": 0.9},
        {"industry": "real_estate", "correlation": 0.7}
      ],
      "tech": [
        {"industry": "communication", "correlation": 0.8},
        {"industry": "consumer", "correlation": 0.6}
      ]
    }
  }
}
```
