
# AI股票分析助手 全新设计方案（专为小白散户打造）
## 一、产品核心定位
### 1.1 定位说明
专为零金融基础的小白散户打造的极简AI股票分析助手，不需要懂任何专业的股票知识，只需要简单操作就能获得精准的选股建议、专业的个股分析和持仓优化指导，帮助普通人轻松理财，规避风险，提高投资收益。
### 1.2 核心用户
- 核心用户：没有专业金融知识的小白散户投资者
- 用户画像：20-45岁，有理财需求但不懂股票分析，平时没有时间研究股票，经常跟风买股票踩坑，需要简单直接的投资建议
### 1.3 核心价值
- 零门槛：不需要懂任何专业金融知识，操作简单，一看就会
- 够直接：所有输出全是大白话，直接给明确的「买入/持有/卖出」建议
- 够精准：基于A股市场特点优化的AI分析，选股和分析准确率远高于普通人自己研究
- 省时间：不需要花大量时间研究股票、看新闻、看财报，AI帮你全部搞定
## 二、核心功能设计（按优先级排序）
### 2.1 最高优先级：AI智能选股（新增核心功能，满足精准选股需求）
#### 功能说明
用户只需要简单设置筛选条件，AI自动从全市场5000多只A股股票中筛选出最符合条件的优质股票，给出明确的推荐列表和分析。
#### 操作流程
1. 用户选择或输入筛选条件：
   - 必选：风险偏好（保守/稳健/激进）
   - 可选：感兴趣的行业（比如消费、科技、医疗、新能源等，可以多选）、期望收益目标（比如年化5%/10%/20%+）、可承受最大回撤（比如5%/10%/20%）
   - 支持自然语言输入：比如直接输入「我想买稳一点的，风险低的股票」、「帮我选最近热门的科技股」
2. 系统自动处理：
   - 采集全市场所有股票的最新基础数据、财务数据、技术指标、新闻舆情数据
   - 按照用户的筛选条件和风险偏好，用AI综合评分筛选
   - 排除ST股、风险股、业绩暴雷股
3. 输出结果：
   - 推荐Top10股票列表，按综合评分从高到低排序
   - 每只股票包含：股票名称+代码、当前价格、综合评分、推荐理由（大白话，100字以内）、明确操作建议（买入/持有/观察）、风险提示（大白话，最多2条）
   - 整体推荐说明：本次选股的整体风格、适合的人群、注意事项
#### 额外功能
- 支持自定义专业筛选：如果用户懂一点专业知识，可以自定义市盈率、ROE、毛利率、市值、成交量等指标筛选
- 支持相似股票对比：点击任意推荐股票，可以查看同行业3只相似股票的对比分析，帮用户选到最优的
### 2.2 第二优先级：AI个股分析（原有功能大幅优化，小白友好）
#### 功能说明
输入股票代码/名称，一键生成完整的AI分析报告，所有内容全是大白话，不需要懂专业知识就能看懂，直接给明确的操作建议。
#### 输出内容（全小白友好优化）
1. 基础信息卡片：股票名称、代码、当前价格、涨跌幅、所属行业、市值
2. 四项评分（每项都有大白话解释）：
   - 行业评分（0-100）：解释「这个行业目前处于什么阶段，是景气还是低迷」
   - 财务评分（0-100）：解释「这家公司赚钱能力怎么样，有没有暴雷风险」
   - 技术评分（0-100）：解释「目前这个价格是高还是低，短期走势怎么样」
   - 舆情评分（0-100）：解释「最近这家公司的新闻是利好还是利空」
   - 综合评分（0-100）：整体评价
3. 核心分析结论（150字以内，大白话，无专业术语）
4. 明确操作建议：直接给出「买入/持有/卖出」结论，附带简单易懂的理由
5. 风险提示：至少3条，全是大白话，明确告诉用户可能的风险
6. 同行业对比：自动推荐3只同行业相似股票对比，帮用户判断是不是最好的选择
### 2.3 第三优先级：AI持仓分析（原有功能优化，更实用）
#### 功能说明
用户导入自己的持仓，一键生成完整的持仓分析报告，给出明确的优化建议，帮用户规避风险，提高收益。
#### 操作流程
1. 用户导入持仓：
   - 支持手动逐个输入：股票名称/代码、持仓数量、买入成本
   - 支持Excel批量导入：直接上传持仓表，自动识别
2. 系统自动分析：
   - 计算持仓总市值、总盈亏、总盈亏率、单只股票盈亏
   - 分析持仓的行业分布、风险集中度
   - 逐个分析每只持仓的质量，给出操作建议
   - 整体评估持仓的风险等级
3. 输出结果：
   - 持仓概览：总市值、总盈亏、盈亏率、风险等级（低/中/高）
   - 行业分布饼图：直观展示各个行业占比，提示是否过于集中
   - 单只持仓分析：每只股票的当前分析、操作建议
   - 优化建议：明确的调整建议，比如「科技行业占比80%，风险太集中，建议减仓20%到消费行业分散风险」、「XX股票基本面恶化，建议卖出」，全是大白话，一看就知道怎么操作
### 2.4 辅助功能（轻量实用）
#### 2.4.1 我的关注
- 支持收藏关注的股票
- 自动每天更新关注股票的最新分析结果和价格变动
- 有重大变动时（比如发布重要公告、业绩暴雷、涨跌超过阈值）自动提醒
#### 2.4.2 历史报告
- 所有历史选股报告、个股分析报告、持仓分析报告永久保存
- 支持按时间、股票名称搜索
- 支持导出为PDF/图片分享
#### 2.4.3 小白课堂
- 简单的股票基础知识科普，新手教程
- 常见问题解答
- 理财小知识分享，边用边学
## 三、系统架构设计（基于原有成熟架构优化，稳定可靠）
### 3.1 整体架构
保持原有架构的优点，新增智能选股相关模块，架构更简洁，开发速度快，维护成本低：

```
├─ 智能选股引擎模块（新增）
├─ 个股分析模块
├─ 持仓分析模块
├─ 数据采集模块
├─ AI分析模块（对接大模型）

├─ 关注列表模块

└─ 定时任务模块（APScheduler）
↓

↓
外部接口
├─ AkShare（免费A股数据）
├─ 财经新闻爬虫

```
### 3.2 核心模块说明
#### 3.2.1 新增：智能选股引擎
- 每日凌晨自动批量更新全市场所有股票的最新数据（基础信息、财务数据、技术指标、新闻）
- 自动预处理数据，计算各项评分，提前筛选掉ST股、风险股、业绩亏损股
- 支持多维度快速筛选，根据用户的条件实时返回最优的股票列表
#### 3.2.2 优化：AI分析模块
- 新增小白话转换层：所有大模型返回的专业内容自动转换成大白话，自动解释专业术语
- 新增选股专用Prompt：针对选股场景优化的Prompt，确保推荐的股票符合A股市场特点，风险低，收益稳定
- 新增持仓分析专用Prompt：针对持仓优化场景优化的Prompt，给出的建议更实用，更符合小白用户的需求
#### 3.2.3 优化：数据采集模块
- 每日自动全量更新所有A股股票的最新数据
- 支持批量采集多只股票的数据，提高选股效率
- 新增数据质量校验，确保采集的数据准确无误
### 3.3 技术栈（和原有设计一致，成熟稳定，开发速度快）
- 前端：Vue3 + JavaScript + Element Plus + ECharts（PC端优先，简单易用）
- 后端：Python + FastAPI（高性能，原生异步支持，自动生成API文档）

- 定时任务：APScheduler（轻量简单，不需要中间件，适合个人项目）
- 数据来源：AkShare（免费开源A股数据） + 东方财富/同花顺新闻爬虫（免费）
### 3.4 部署方案

- 后续可升级云部署：如果需要随时随地访问，可以部署到云服务器，每月成本几十元
## 四、AI分析优化设计（专为小白用户和A股市场优化）
### 4.1 Prompt优化
- 所有Prompt都针对A股市场特点优化，加入A股特有的交易规则、监管制度、市场风格说明
- 强制要求大模型输出大白话，禁止使用专业术语，如果必须使用要附带解释
- 强制要求给出明确的「买入/持有/卖出」建议，不能模棱两可
- 加入示例输出，确保大模型返回的格式统一，符合要求
### 4.2 输出格式规范
所有输出都使用Pydantic模型定义，确保结构化返回，前端展示友好：
```python
# 股票推荐结果
class StockRecommendation(BaseModel):
    stock_code: str
    stock_name: str
    current_price: float
    overall_score: float
    reason: str  # 推荐理由，大白话，100字以内
    advice: str  # 操作建议：买入/持有/观察
    risk_tips: List[str]  # 风险提示，大白话，最多2条
# 个股分析结果
class StockAnalysisResult(BaseModel):
    basic_info: dict  # 基础信息
    industry_score: float
    industry_score_explain: str  # 行业评分大白话解释
    financial_score: float
    financial_score_explain: str  # 财务评分大白话解释
    technical_score: float
    technical_score_explain: str  # 技术评分大白话解释
    sentiment_score: float
    sentiment_score_explain: str  # 舆情评分大白话解释
    overall_score: float
    analysis_summary: str  # 分析结论，大白话，150字以内
    investment_advice: str  # 操作建议：买入/持有/卖出 + 大白话理由
    risk_tips: List[str]  # 风险提示，大白话，至少3条
    similar_stocks: List[dict]  # 相似股票对比
# 持仓分析结果
class PortfolioAnalysisResult(BaseModel):
    total_value: float
    total_profit: float
    total_profit_rate: float
    risk_level: str  # 低/中/高
    industry_distribution: Dict[str, float]
    risk_concentration_explain: str  # 风险集中度大白话说明
    stock_analysis: List[dict]  # 单只持仓分析
    suggestions: List[str]  # 优化建议，大白话，每条都是可直接操作的
    overall_assessment: str  # 综合评价
```
### 4.3 准确率提升策略
- 结构化输入：所有输入数据都整理成结构化的表格形式，减少大模型理解错误
- 结果校验：对大模型返回的评分、建议进行合理性校验，明显不合理的自动重新调用
- 缓存机制：相同股票的分析结果缓存24小时，相同选股条件的结果缓存12小时，减少重复调用，提高响应速度，节省成本
