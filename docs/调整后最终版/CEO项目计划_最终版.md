
# AI股票分析助手 CEO项目计划（最终版）
## 一、项目基本信息
- 项目定位：为小白散户打造的零门槛AI股票分析工具
- 核心目标：实现精准选股、大白话个股分析、持仓风险评估，帮用户避免跟风踩坑，提升投资收益
- 开发模式：理想架构版，模块化插件化设计，支持后续长期迭代扩展
- 总开发周期：2周（14个工作日）
- 部署方式：优先本地部署，后续可升级云部署

## 二、最终开发范围
### ✅ 核心功能（原方案B包含）
1. **系统架构**：前后端分离，模块化插件化设计，预留自动盯盘、自动交易、多数据源扩展点
2. **数据采集模块**：对接AkShare免费A股数据，支持定时全量更新和盘中行情更新
3. **AI分析模块**：对接豆包大模型（支持切换通义千问/本地Ollama模型），实现股票分析、持仓分析功能，输出大白话结构化结果
4. **核心业务模块**：
   - 用户管理：注册登录、API密钥配置
   - 持仓管理：手动/批量导入持仓，持仓分析
   - 关注列表：关注股票，基础监控
   - 分析报告管理：历史报告保存、查看、导出
5. **前端**：基于现有demo改造，所有功能页面，可视化展示分析结果
6. **基础能力**：缓存机制、异常处理、请求重试、日志监控

### ✅ 扩展功能（用户选择加入本次开发）
1. **回测功能**：支持用历史数据验证AI选股和量化策略的收益率，直观看到选股效果
2. **多渠道通知功能**：对接飞书机器人通知，预留微信/邮件/短信扩展点，触发提醒条件时自动发送通知
3. **自定义提醒规则功能**：支持灵活配置提醒条件（价格、涨跌幅、PE、ROE、持仓盈亏等），界面可视化配置，不需要改代码
4. **内置量化选股策略功能**：内置4种经典量化选股策略（低PE高ROE、高股息率、小盘成长、行业龙头），可直接使用，支持和AI选股结果对比
5. **持仓收益走势图表功能**：自动记录每日持仓收益，生成收益走势曲线，展示最大回撤、年化收益率等投资指标，方便复盘

### ❌ 本次不开发（后续迭代可加）
- 帮助中心模块
- 用户反馈模块
- 市场板块分析模块
- 自动交易功能
- 移动端APP

## 四、成本说明
### 初期零成本
1. 数据成本：AkShare开源免费，新闻爬虫免费
2. AI成本：豆包大模型个人免费额度每月100万token，足够每天分析100只股票使用，完全满足需求
3. 服务器成本：本地部署不需要服务器成本
### 后续成本极低
- 如果使用付费大模型，每月成本不超过50元
- 如果升级云部署，最低配服务器每月成本30-50元

## 六、风险控制
1. **大模型调用失败**：加重试机制，支持多模型 fallback，一个模型调用失败自动切换到其他模型
2. **数据采集失败**：加异常捕获，失败自动重试，多数据源备用
3. **开发延期风险**：核心功能优先开发，非核心功能可以放到后续迭代，确保2周内交付可用版本
4. **使用复杂度高**：所有功能操作简化，大白话提示，编写详细的使用文档，确保零门槛使用
