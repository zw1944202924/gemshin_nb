# 用户操作案例

## 11.1 案例背景
**用户角色**：个人投资者
**目标**：分析阿里巴巴(BABA)股票并决定是否投资
**场景**：用户想了解阿里巴巴的最新表现，获取AI分析建议，并查看相关板块表现

## 11.2 操作流程

### 步骤1：登录系统
1. 用户访问系统登录页面
2. 输入用户名和密码
3. 点击"登录"按钮
4. 系统验证身份后跳转到仪表盘页面

### 步骤2：查看市场概览
1. 在仪表盘页面，用户查看市场概览卡片
2. 观察主要指数走势
3. 查看个人持仓概览和关注列表提醒
4. 浏览最近分析报告摘要和市场热点新闻

### 步骤3：分析阿里巴巴股票
1. 点击左侧菜单栏的"股票分析"选项
2. 在顶部搜索栏输入"BABA"或"阿里巴巴"
3. 从搜索结果中选择"阿里巴巴 (BABA)"
4. 查看左侧的基本信息、K线图表和技术指标
5. 切换不同时间周期查看K线走势
6. 查看基本面分析和舆情分析
7. 打开右侧AI分析窗口，查看综合评分和投资建议
8. 在智能问答框中输入"阿里巴巴未来6个月走势如何？"
9. 查看AI的回答和预测分析

### 步骤4：管理关注列表
1. 点击左侧菜单栏的"关注列表"选项
2. 点击"添加股票"按钮
3. 输入"BABA"并添加到关注列表
4. 为阿里巴巴设置价格提醒：上限90美元，下限80美元
5. 设置技术指标提醒：MACD金叉提醒

### 步骤5：查看板块分析
1. 点击左侧菜单栏的"市场板块"选项
2. 在板块列表中选择"科技"板块
3. 查看科技板块的历史表现趋势
4. 查看科技板块的龙头股列表
5. 分析板块间相关性

### 步骤6：查看分析报告
1. 点击左侧菜单栏的"分析报告管理"选项
2. 在报告列表中找到阿里巴巴的分析报告
3. 点击查看报告详情
4. 点击"导出PDF"按钮，下载完整分析报告

### 步骤7：设置通知
1. 点击左侧菜单栏的"通知中心"选项
2. 确认价格提醒和技术指标提醒已开启
3. 查看已有的通知记录

## 11.3 预期结果
- 用户获得了阿里巴巴股票的全面分析
- 用户了解了科技板块的整体表现
- 用户设置了股票提醒，以便及时获取价格和技术指标变化
- 用户下载了完整的分析报告，用于后续参考
- 用户对投资决策有了更清晰的依据
