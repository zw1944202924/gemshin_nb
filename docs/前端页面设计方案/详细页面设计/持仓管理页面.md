# 持仓管理页面

## 布局
- **布局**：左侧持仓列表和图表分析，右侧AI分析窗口

## 功能
- 持仓列表（股票、持仓数量、成本价、当前价、盈亏）
- 持仓详情（点击查看单个股票详细信息）
- 投资组合分析（行业分布、风险评估、收益分析）
- 交易记录（买入、卖出历史）
- 再平衡建议
- AI分析窗口：
  - 投资组合整体评估
  - 风险分散分析
  - 行业配置建议
  - 智能调仓建议
  - 收益预测分析
  - 市场热点与持仓相关性分析

## 设计要点
- 表格支持排序、筛选和分页
- 盈亏数据使用颜色区分（红涨绿跌或绿涨红跌，可配置）
- 投资组合分析使用雷达图和饼图
- 交易记录时间轴展示
- AI分析窗口固定在右侧，可折叠/展开
- 支持批量操作（如批量添加到关注列表）
