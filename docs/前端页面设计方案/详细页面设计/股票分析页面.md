# 股票分析页面

## 布局
- **布局**：顶部搜索栏，左侧股票信息和分析工具，右侧AI分析窗口

## 功能
- 股票搜索和选择
- 基本信息展示（代码、名称、当前价格、涨跌幅）
- K线图表（支持多种周期切换）
- 技术指标分析（MACD、KDJ、RSI等）
- 基本面分析（财务数据、估值指标）
- 舆情分析（新闻情感、社交媒体情绪）
- AI分析窗口：
  - 综合评分和投资建议
  - 智能问答功能（用户可直接向AI提问关于该股票的问题）
  - 预测走势分析
  - 风险评估
  - 相似股票推荐

## 设计要点
- K线图支持缩放和拖拽
- 技术指标可自定义显示
- 财务数据表格支持排序和筛选
- AI分析窗口固定在右侧，可折叠/展开
- 智能问答支持实时反馈
- 支持添加到关注列表和持仓
