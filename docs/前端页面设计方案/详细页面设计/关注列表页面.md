# 关注列表页面

## 布局
- **布局**：左侧关注股票列表和监控设置，右侧AI分析窗口

## 功能
- 关注股票列表（当前价格、涨跌幅、触发状态）
- 价格提醒设置（设置价格上下限）
- 技术指标提醒（如金叉死叉、突破压力位等）
- 新闻提醒（重要新闻推送）
- 提醒历史记录
- AI分析窗口：
  - 关注列表整体市场表现分析
  - 潜在机会与风险预警
  - 行业热点分析
  - 关注股票相关性分析
  - 智能提醒建议（基于历史数据和市场趋势）
  - 个性化推荐（基于关注列表的投资偏好）

## 设计要点
- 列表支持排序和分组
- 提醒设置使用直观的滑块和输入框
- 触发状态使用醒目的图标和颜色
- AI分析窗口固定在右侧，可折叠/展开
- 支持批量设置提醒
- 提醒历史记录可筛选和搜索
